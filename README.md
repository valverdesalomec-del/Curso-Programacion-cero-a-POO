#BOOTCAMP PROGRAMACION DE CERO A POO
## Descripcion
Este repositorio corresponde al bootcamp *Programacion de Cero a POO*, un curso virtual en vivo orientado a estudiantes que estan iniciando en programacion y deseando construir una base solida antes de avanzar hacia Programacion Orienteada a Objetos

El enfoque del bootcamp sigue una progresion academica practica:
**logica de programacion -> diagramas -> pseudocodigo -> C++ estructurado -> Java POO**.

A lo largo del curso, se trabaja en clases en vivo, ejercicios practicos, tareas autonomas, evaluaciones y proyectos integradores, manteniendo siempre una metodologia centrada en comprender primero la logica, luego representarla y finalmente impletarla en codigo.

## Objetivo del Repositorio
Este respositorio tiene como finalidad organizar y almacenar el material, ejercicios, practicas y proyectos desarollados durante el bootcamp, ademas de ser como evidencia del proceso de aprendizaje mediante el uso de Git y GitHub.

## Contenido del curso
Hasta este punto del bootcamp, se ha trabajo temas como:

- logica y resolucion de problemas.
- diagramas de flujo
- pseudocodigo en Pseint
- introduccion a C++
- variables, operadores y entrada/salida
- estructuras y condiciones
- estructuras repetitivas
- arreglos
- funciones y modularizacion
- depuracion y correccion de errores
- introduccion a Git y GitHub
- preparacion del mini Proyecto en C++
